# Virtual Drum Kit

A web-based interactive drum kit that allows users to play various drum sounds using their keyboard or by clicking the on-screen buttons. This project explores **DOM manipulation**, **Event Listeners**, and **Audio Playback** in JavaScript.

## Features
* **Keyboard Support:** Play sounds using the keys: `W`, `A`, `S`, `D`, `J`, `K`, and `L`.
* **Click Interaction:** Each drum pad is clickable for those who prefer using a mouse.
* **Visual Feedback:** Buttons animate when triggered to provide a responsive user experience.
* **High-Quality Audio:** Distinct sounds for snare, tom-toms, kick drum, and crash cymbal.

---

## Tech Stack
* **HTML5:** Structured the drum pads and layout.
* **CSS3:** Styled the interface with a dark theme, custom fonts, and button animations.
* **JavaScript (ES6):** Handled event listeners for both `keydown` and `click` events to trigger audio.

## Concepts Used
* Adding **Event Listeners** to multiple elements using loops.
* Using the **Audio object** in JavaScript to play sounds programmatically.
* Implementing **Switch statements** to handle different key inputs efficiently.
* Applying and removing CSS classes dynamically to create button animations.
